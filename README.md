# Aysel Agency — الموقع الرسمي

موقع شركة Aysel Agency: وكالة رقمية متكاملة (برمجة، تصميم، تسويق، ذكاء اصطناعي).

## التقنيات

- **Next.js 16** (App Router + Turbopack) + TypeScript
- **Tailwind CSS 4** — نظام ألوان كحلي + ذهبي
- **Framer Motion** — أنيميشن احترافي (Scroll reveals, counters, marquee, carousel)
- **next-intl** — ثنائي اللغة: عربي (RTL، افتراضي) + إنجليزي
- **Supabase** — قاعدة البيانات، المصادقة، وتخزين الصور

## الصفحات

| المسار | الوصف |
|--------|-------|
| `/ar` و `/en` | الصفحة الرئيسية (Hero، الأقسام الأربعة، من نحن، إحصائيات، أعمالنا، منهجيتنا، آراء العملاء، تواصل) |
| `/admin` | لوحة التحكم (تتطلب تسجيل دخول) |
| `/admin/login` | تسجيل الدخول |

## لوحة التحكم

مربوطة بـ Supabase وتدير:

- **الرسائل** — رسائل نموذج التواصل (قراءة/حذف)
- **الأعمال** — معرض الأعمال مع رفع صور (CRUD كامل)
- **الأقسام والخدمات** — إضافة/إخفاء/حذف خدمات كل قسم
- **الإعدادات** — بيانات التواصل، روابط السوشال ميديا، أرقام الإحصائيات

## قاعدة البيانات (Supabase)

المشروع: `ayselagency` (`embgaxvyeekbcazcxnwx`)

الجداول: `divisions`, `services`, `projects`, `messages`, `testimonials`, `site_settings` — جميعها محمية بـ RLS (قراءة عامة للمحتوى المنشور، كتابة للمدير المسجل فقط، وإدخال رسائل للجميع). صور الأعمال تُرفع إلى bucket باسم `media`.

> إذا تعذّر الوصول لقاعدة البيانات أثناء البناء، الموقع يعرض محتوى احتياطياً مدمجاً (`src/lib/fallback-data.ts`) مطابقاً للبيانات الأولية.

## التشغيل محلياً

```bash
npm install
npm run dev
```

## النشر (Vercel)

1. اربط المستودع بـ Vercel.
2. لا حاجة لمتغيرات بيئة — مفاتيح Supabase العامة (publishable) مدمجة في `src/lib/config.ts` ويمكن تجاوزها عبر `NEXT_PUBLIC_SUPABASE_URL` و `NEXT_PUBLIC_SUPABASE_ANON_KEY` (انظر `.env.example`).
3. اربط الدومين `ayselagency.com`.
